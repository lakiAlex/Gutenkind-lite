== Description ==

The Gutenkind Lite is a no fuss, stunning WordPress theme built to power websites for creative agencies, freelancers, artists, designers, photographers and those who love to create.
It's lightweight, simple to use and powerful in features.
Featuring a purposeful, responsive design and a fully-featured WooCommerce shop. You can get a professional blog & shop website up and running effortlessly, with zero coding needed. The theme will make your website look fantasic.

The GPLv2+

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type 'Gutenkind Lite' in the search form and press the 'Enter'.
3. Click on the 'Activate' button to use your new theme right away.
5. Navigate to Appearance > Customize in your admin panel and customize to taste.
